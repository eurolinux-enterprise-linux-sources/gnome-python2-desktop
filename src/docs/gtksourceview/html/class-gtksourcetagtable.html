gtksourceview.SourceTagTable.add_tags

    def add_tags(tags)

tags :

 a list conteining gtksourceview.SourceTag objects.

The set_check_brackets() method Add a list of tag to the table. The added tags are assigned the highest priority in the table. If a tag is already present in table or has the same name as an already-added tag, then it is not added to the table.

gtksourceview.SourceTagTable.remove_source_tags

    def remove_source_tags()

The remove_source_tags() method removes all the source tags from the table. This will remove the table's reference to the tags, so be careful - tags will end up destroyed if you don't have a reference to them.
